How to run:
-----------
1- Go to the root folder where package.json is
2- run "Npm Install"
3- run "Npm Start"
4- The project will be ready on localhost:3000


How to test:
------------
1- Go to the root folder
2- run "Npm test"


What is included:
-----------------
1- Hooks
2- Jest tests
3- Error Boundary
4- Css variables
5- Dark/Light Themes
6- PropTypes


What is not included:
---------------------
1- Typescript
2- Hook tests
3- Carousel is not created (used one off the shelf component)
